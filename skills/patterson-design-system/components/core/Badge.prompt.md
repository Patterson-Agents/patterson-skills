**Badge** — compact uppercase status/category label. Soft-tint by default; pass `solid` for a filled chip.

```jsx
<Badge tone="teal">In stock</Badge>
<Badge tone="sky" solid>New</Badge>
<Badge tone="danger">Backordered</Badge>
```

Tones: `navy`, `sky`, `green`, `teal`, `purple`, `gray`, `warning`, `danger`.
