**IconButton** — circular, icon-only control for toolbars, nav and cards. Always pass a `label` for accessibility.

```jsx
<IconButton label="Search" variant="subtle"><SearchIcon/></IconButton>
<IconButton label="Menu" variant="ghost"><MenuIcon/></IconButton>
```

Variants: `solid` (navy), `sky`, `ghost`, `subtle` (navy-10 wash). Sizes `sm`/`md`/`lg` → 34/44/54px.
