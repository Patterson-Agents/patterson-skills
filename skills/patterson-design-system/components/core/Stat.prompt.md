**Stat** — oversized statistic with a supporting label, the signature device on Patterson marketing pages.

```jsx
<Stat value="86M" label="U.S. households have pets" tone="sky" />
<Stat value="98%" label="of customers reached in 1–2 days" tone="navy" align="center" />
```

Tones: `sky`, `navy`, `teal`, `green`. Keep labels to a phrase.
